## Image Variants/Tags

### `focal`, `2004`

- Ubuntu 20.04 (focal)
- PHP 5.6 to PHP 7.4
- Node.js 12.18.3
- Yarn 1.22.4

### `latest`, `bionic`, `1804`

- Ubuntu 18.04 (bionic)
- PHP 5.6 to PHP 7.4
- Node.js 12.18.3
- Yarn 1.22.4

### `xenial`, `1604`

- Ubuntu 16.04 (xenial)
- PHP 5.6 to PHP 7.4
- Node.js 12.18.3
- Yarn 1.22.4

### `focal-slim`, `2004-slim`

- Ubuntu 20.04 (focal)
- Node.js 12.18.3
- Yarn 1.22.4

### `latest-slim`, `bionic-slim`, `1804-slim`

- Ubuntu 18.04 (bionic)
- Node.js 12.18.3
- Yarn 1.22.4

### `xenial-slim`, `1604-slim`
